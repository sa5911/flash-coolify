# Client Module

## Overview
The Client module manages client relationships, contracts, and service sites (specifically for security services).

## Key Components
- **Clients**: Client profiles and contact details.
- **Sites**: Locations where services are provided.
- **Contracts**: Service agreements and billing terms.
- **Guard Allocation**: Assigning security personnel to client sites.

## Key Models
- `Client`: Customer details.
- `ClientSite`: Service locations.
- `ClientContract`: Billing and service agreements.
