# Finance Module

## Overview
The Finance module handles the company's financial transactions, expense tracking, and account management.

## Key Components
- **Expenses**: Tracking operational expenses.
- **Accounts**: Management of chart of accounts.
- **Journal Entries**: Double-entry bookkeeping records.
- **Payroll Integration**: Integration with HR for salary payments.

## Key Models
- `Expense`: Expense records.
- `FinanceAccount`: Chart of accounts.
- `FinanceJournalEntry`: General ledger entries.
