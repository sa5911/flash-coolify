# Fleet Module

## Overview
The Fleet module manages the company's vehicle fleet, including vehicle tracking, maintenance logs, and fuel consumption.

## Key Components
- **Vehicles**: Registry of all company vehicles.
- **Fuel**: Tracking fuel entries and mileage efficiency.
- **Maintenance**: Logging repairs and scheduled maintenance.
- **Assignments**: Assigning vehicles to drivers or departments.

## Key Models
- `Vehicle`: Vehicle details (Reg No, Model, etc.).
- `FuelEntry`: Fuel transaction logs.
- `VehicleMaintenance`: Repair history.
