# HR Module

## Overview
The HR module manages employee records, attendance, and payroll. It is designed to handle complex payroll calculations, leave management, and employee document tracking.

## Key Components
- **Employees**: Management of employee profiles (`Employee2`).
- **Attendance**: Tracking of daily attendance, leave, and overtime.
- **Payroll**: Automated salary calculation, tax deduction, and payslip generation.
- **Analytics**: HR reporting and statistics.

## Key Models
- `Employee2`: The main employee record.
- `AttendanceRecord`: Daily logs.
- `PayrollSheetEntry`: Monthly salary details.
