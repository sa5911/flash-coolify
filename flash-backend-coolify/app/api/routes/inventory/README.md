# Inventory Module

## Overview
The Inventory module manages company assets, including general office supplies and restricted items (security equipment).

## Key Components
- **General Inventory**: Tracking of consumable and non-consumable office items.
- **Restricted Inventory**: Specialized tracking for sensitive items (e.g., security gear).
- **Assignments**: Issuing items to employees and tracking returns.
- **Stock Management**: Stock in/out and balance tracking.

## Key Models
- `GeneralItem`: Standard inventory items.
- `RestrictedItem`: Sensitive inventory items.
- `InventoryAssignment`: Tracking who has what item.
