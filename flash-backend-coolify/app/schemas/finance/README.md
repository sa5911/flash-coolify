# Schemas - Finance

## Overview
Pydantic schemas for financial transactions and accounting.

## Schemas
- **ExpenseBase**, **ExpenseCreate**, **Expense**: Expense tracking
- **FinanceAccountBase**, **FinanceAccount**: Chart of accounts
- **JournalEntryBase**, **JournalEntry**: General ledger
- **PayrollSheetEntryBase**, **PayrollSheetEntry**: Salary calculations

## Usage
Used for expense management, accounting, and payroll processing APIs.
