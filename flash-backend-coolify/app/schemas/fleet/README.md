# Schemas - Fleet

## Overview
Pydantic schemas for vehicle fleet management.

## Schemas
- **VehicleBase**, **VehicleCreate**, **Vehicle**: Vehicle registry
- **FuelEntryBase**, **FuelEntry**: Fuel transactions
- **MaintenanceBase**, **Maintenance**: Repair logs
- **VehicleAssignmentBase**, **VehicleAssignment**: Driver assignments

## Usage
Used for fleet management, fuel tracking, and maintenance scheduling APIs.
