# Schemas - HR

## Overview
Pydantic schemas for HR and payroll operations.

## Schemas
- **Employee2Base**, **Employee2Create**, **Employee2**: Employee profiles
- **AttendanceRecordBase**, **AttendanceRecord**: Daily attendance
- **LeaveRequestBase**, **LeaveRequest**: Leave applications
- **PayrollSheetEntryBase**, **PayrollSheetEntry**: Salary details

## Usage
Used for employee management, attendance tracking, and payroll APIs.
